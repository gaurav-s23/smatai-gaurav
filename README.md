# SMAT.AI
AI-powered predictive maintenance system with automated short &amp; long report generation.
Description
MaintAI is a predictive maintenance system that leverages AI to monitor machine health, predict potential failures, and automatically generate both short and long maintenance reports.
It helps industries reduce downtime, optimize maintenance schedules, and make data-driven decisions.

Features
Upload CSV datasets containing machine sensor readings
Automatic preprocessing and encoding of categorical data
Train a Random Forest model to predict machine failures
Generate short, concise maintenance reports for quick insights
Generate detailed, long reports with analysis and recommendations
Download all reports as a CSV file
View a random report in the app for quick inspection
Steps inside the app:

Upload your CSV dataset using the sidebar.

View a preview of raw data.

Train and evaluate the predictive model.

Generate short and long maintenance reports.

Download all reports as a CSV or view a random report for quick inspection.

Sample Data

Include a small sample_data.csv to test the app. Columns should include: Temperature, Pressure, Vibration, Machine failure, and optional operator_note.
